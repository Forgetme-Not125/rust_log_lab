# 迭代版本说明

本项目在基础版本“多线程日志分析 CLI 工具”的基础上，新增了 `export.rs` 报告导出模块。

## 新增功能

- 支持将分析结果导出到本地文件。
- 新增 Markdown 报告格式。
- 当用户使用 `--out report.md`、`--out report.json`、`--out report.csv` 时，程序可以根据扩展名自动推断输出格式。
- 报告导出逻辑从命令行模块中拆分出来，使模块职责更加清晰。

## 示例命令

```bash
cargo run -- analyze examples/sample.log --out report.md
cargo run -- analyze examples/sample.log --format markdown --out report.md
cargo run -- analyze examples/sample.log --out report.json
cargo run -- analyze examples/sample.log --out report.csv
```

## 设计意义

基础版本只能在终端查看分析结果，迭代版本可以把结果保存下来，方便归档、提交或后续查看。该模块体现了 Rust 的文件操作、Result 错误处理、trait 抽象和模块化扩展能力。
