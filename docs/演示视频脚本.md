# Rust Log Lab 演示视频脚本（5 分钟以内）

## 0:00 - 0:30 项目介绍
大家好，我的 Rust 期末大作业是 Rust Log Lab，一个多线程命令行日志分析器。它可以读取 key=value 格式日志，统计日志级别、服务调用次数、慢请求和错误记录，并输出 text、json、csv 三种格式报告。

## 0:30 - 1:30 功能演示
先运行：

```bash
cargo run -- demo
```

展示内置 demo 输出。然后运行：

```bash
cargo run -- analyze examples/sample.log
cargo run -- analyze examples/sample.log --level WARN
cargo run -- analyze examples/sample.log --service auth
cargo run -- analyze examples/sample.log --format json --out report.json
```

说明工具支持过滤、统计和导出。

## 1:30 - 3:30 核心实现介绍
打开 src 目录，说明模块划分：

- cli.rs 负责命令行参数解析。
- parser.rs 负责解析 key=value 日志，支持带引号的消息。
- model.rs 定义 LogRecord、LogLevel、OutputFormat 等核心类型。
- analyzer.rs 使用 thread 和 mpsc channel 实现多线程分析，每个线程处理一个分片，最后合并统计结果。
- report.rs 使用 trait 抽象不同报告格式。
- error.rs 使用 AppError 统一错误处理。

重点说明 Rust 特性：所有权转移到线程、借用记录进行统计、enum 表示日志级别、trait 表示解析器和报告渲染器、Result 传播错误。

## 3:30 - 4:30 测试与工程规范
运行：

```bash
cargo test
cargo fmt
cargo clippy
```

说明项目包含 parser、model、cli、analyzer、report 的关键测试。
